# problema-de-monty-hall-vue
Implementando o Problema de Monty Hall usando Vue.js - Curso Web Moderno com JS.

## Faça o clone ou download ZIP do projeto

### Acesse a pasta 'montyhall'
```
cd montyhall
```

### Instale as dependências
```
npm install
```

### Execute a aplicação
```
npm run serve
```

Mais comandos no arquivo README.md dentro da pasta montyhall.
